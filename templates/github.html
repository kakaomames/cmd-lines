GitHub API アップローダー サービス

このエンドポイント /github は **POSTリクエスト** のみを受け付けます。

🚀 POST リクエストの送り方

以下の情報をリクエストに含めてください:

  • HTTPメソッド: POST
  • リクエストヘッダー (必須): REPOFILEPASS (GitHubリポジトリ内の保存パス 例: data/model.obj)
  • リクエストヘッダー (任意): COMMITMESSAGE (コミットメッセージ)
  • リクエストボディ: アップロードしたいファイルの内容 (バイナリデータ)

💡 テスト方法 (例: cURL)

curl -X POST \
  -H "Content-Type: application/octet-stream" \
  -H "REPOFILEPASS: data/test_model_001.obj" \
  -H "COMMITMESSAGE: Flaskアップローダーのテスト" \
  --data-binary @./[ファイル名.obj] \
  https://xeroxapp032.vercel.app/github

※ [ファイル名.obj] は実際に存在するファイル名に置き換えてね。
